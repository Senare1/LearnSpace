Aide - Réinitialisation du mot de passe

Comment réinitialiser votre mot de passe ?

Suivez ces étapes simples pour réinitialiser votre mot de passe :

  1. Visitez notre site web.
  2. Cliquez sur le lien "Mot de passe oublié" ou "Réinitialiser mot de passe" sur la page de connexion.
  3. Renseignez votre adresse e-mail associée à votre compte.
  4. Un e-mail de réinitialisation de mot de passe vous sera envoyé avec un lien unique.
  5. Cliquez sur le lien dans l'e-mail et suivez les instructions pour choisir un nouveau mot de passe sécurisé.

Si vous rencontrez des difficultés pour réinitialiser votre mot de passe, veuillez nous contacter :
